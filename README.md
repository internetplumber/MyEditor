# MyEditor, a simple macOS Text Editor

This is a "small" text editor for macOS that was the result of playing about with coding on (free) Google AI that started with the prompt "Create the smallest possible gui plain text editor for macos using swift. It should have facilities to load, save, copy, paste, search, replace." Then many, many refinements after that so that it built and worked.

It was inspired by Dave Plummer's YouTube video about creating a 2.5kB version of Notepad in Windows by ensuring you use all the facilities that the operating system provides. It is a bit of fun, no more than that. I may come back and look closer at how it was built, but I may not. I may come back and update the documentation to show how to build it, but again, I may not.

Frankly the in-browser GitHub editor I'm using has just as many features and also supports MarkDown, so there is no reason to use this, it is just hacking around.
